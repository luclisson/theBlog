In general spring boot is a toll to  build production ready spring apps quickly without needing to configure or add dependencies all one by one.

- [[spring starter projects]] provide you with the needed dependencies
- all of the dependencies are configured with [[spring boot auto configuration]] based on what frameworks you have added to your project
- [[spring boot dev tools]] improves the dev's productivity
- [[spring boot manage application config using profiles]] can be used to differinciate between the different environments
- there are [[embedded servers spring boot]]
- [[Spring MVC]] simplifies building web apps and rest api's
